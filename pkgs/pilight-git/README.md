# pilight-git

## Description

AUR package for pilight-git

## Installation

``` bash
mkdir /tmp/pilight-git; cd /tmp/pilight-git
curl https://raw.github.com/pschmitt/pilight-git/master/PKGBUILD
makepkg -si
```
